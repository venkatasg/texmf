\section{Introduction}

This sample document demonstrates all the unique features of the \texttt{practical-report} class. The class is designed for research reports and project documentation with careful attention to typography and readability.

\subsection{Typography Features}

The document uses \alert{Equity A} as the main serif font with proportional lowercase numbers by default. When you need lining figures, use the \verb|\liningfont| command: {\liningfont 1234567890}. Notice how numbers in tables and equations automatically use uppercase monospaced numerals.

This subsubsection heading uses small caps styling. You can also use \verb|\rmsc| for {\rmsc Small Caps Text} and \verb|\rmcs| for {\rmcs All Caps Text} with appropriate letter spacing.

\section{Advanced Features}

\subsection{Quotations}

The class loads \texttt{csquotes} for proper quotation handling. Use \verb|\enquote{}| for \enquote{quotations with proper nesting} that automatically adapt to the language settings.

\subsection{Footnotes}

Regular footnotes\footnote{This is a regular footnote with superscript numbering.} work as expected, with custom formatting that aligns the text properly.

\subsection{Microtype Enhancements}

The document uses microtype with protrusion and expansion enabled, providing subtle improvements to text justification and margin alignment, particularly visible with punctuation marks.

\subsection{Spacing Options}

This document was compiled with the \texttt{inch} option, which sets 1-inch margins on all sides and tighter section spacing. To see the default layout (2.7x alphabet width with more generous section spacing), remove the \texttt{inch} option from the document class declaration.

\section{Bibliography Support}

When the \texttt{biblatex} option is enabled, you can use citations. Use \texttt{\string\cite}, \texttt{\string\textcite}, \texttt{\string\parencite}, etc. for citations.

Example citations: \citep{Smith:01} and \citet{Mueller:02}.

 The class configures biblatex with numeric style, maximum citation names set to 1, and urls/dois/eprints disabled for cleaner references.
